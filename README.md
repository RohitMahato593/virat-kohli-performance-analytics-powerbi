# 🏏 Virat Kohli Performance Analytics — Power BI

## 📌 Overview
A Power BI dashboard built to analyze Virat Kohli's
complete international cricket career — covering runs,
centuries, strike rate, opposition analysis & yearly
performance trends across all three formats.

## 🎯 Key Features
- Total runs, centuries & batting average KPIs
- Balls faced trend from 2008 to 2025
- Boundary %, Runs per Match & Strike Rate breakdown
- Runs by opposition country analysis
- Year-wise & format-wise (ODI/T20I/Test) run trends
- Interactive format slicers
